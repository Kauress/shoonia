GAME CODING CAMP

Ages 10 - 14

1 week, 1 hour everyday. January 18 - 24

Build your own world in 3D

Build a platformer game

Learn and create by coding games

One week full of fun and excitement! You will build your own world and platformer Obby course game using Roblox Studio. Roblox is an easy to use and learn sandbox game enviornment that encourages learning and creativity.
In this game camp, students will learn about Roblox, one of the fastest growing computer game creation platforms. They will learn the basics of Roblox Studio, how to script using Lua and various aspects of Lua programming, design a customized obstacle course (Obby), and finally build their own game world.

Some of the topics you will learn about:

  • 3D objects
  • Variables, conditionals and loops
  • Objects in programming
  • Coding game logic
  • More!

There are no pre-requisites required. All code logic will be explained line by line
